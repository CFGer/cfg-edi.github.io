Make your first page responsive

  • Add Bootstrap to your page

    To add an external stylesheet to your simple index.html file:

    • Make sure you've downloaded Bootstrap, and added the folders to your repository
    • Make sure you've linked the Bootstrap CSS in the <head> of your page.
    • <link rel="stylesheet" href="style.css">.
    • You'll want to start with trying to integrate the Bootstrap grid system as shown in the slides, and other responsive elements you fancy. Refer to the Sample Sites here, and also to the Bootstrap Documentation for help.
    • This documentation can be very overwhelming, so please let us know if you get stuck!
